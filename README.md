# Echoes of Giants

## About

A game that I would consider a shell for what could be a much larger game.

### Controls

#### Keyboard Controls

* Movement
  * WASD

* Shoot
  * Left Mouse Button
* Toggle Aiming Cursor
  * C

#### Xbox 360 Controller Controls

* Movement
  * Left Analog Stick
* Shoot
  * Right Analog Stick
